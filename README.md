# V1-Discord
Ever wanted a funny SAM-based TTS to spam you
with random lines in a voice call? Well now you can
have that!

This bot is only verified on a Windows machine. Use
on other setups at your own discretion.

## Usage
Install `discord.py[voice]`: https://discordpy.readthedocs.io/en/latest/intro.html

Put your discord bot token from https://discord.com/developers/applications into
the `bot.run('Bot token here')` line in `basic_voice.py`.

Run `basic_void.py`! Make sure the bot has the correct
OAuth permissions.
